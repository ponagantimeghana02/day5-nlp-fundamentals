# Sentiment Analysis Report

## Objective

The objective of this task is to analyze the sentiment of text reviews and classify them as Positive, Negative, or Neutral.

## Methodology

The TextBlob library was used to calculate the polarity score of each review.

Polarity Range:

* Positive: Polarity > 0
* Negative: Polarity < 0
* Neutral: Polarity = 0

## Results

| Review                      | Sentiment |
| --------------------------- | --------- |
| This course is amazing      | Positive  |
| The training is excellent   | Positive  |
| The project is difficult    | Negative  |
| The application is terrible | Negative  |

## Summary

* Positive Reviews: 2
* Negative Reviews: 2
* Neutral Reviews: 0

## Conclusion

The sentiment analysis successfully classified the reviews based on their polarity scores. Positive words such as "amazing" and "excellent" resulted in positive sentiment, while negative words such as "difficult" and "terrible" resulted in negative sentiment. Sentiment analysis is widely used in customer feedback systems, product reviews, social media monitoring, and opinion mining.
